# Fonts for Vertera App

This directory contains the font files used in the Vertera app.

## Included Fonts

- Poppins-Regular.ttf
- Poppins-Medium.ttf
- Poppins-SemiBold.ttf
- Poppins-Bold.ttf

## Usage

These fonts are loaded in the root layout file and made available throughout the app.
